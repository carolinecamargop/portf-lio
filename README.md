# Automação de Conteúdo para Redes Sociais com IA

Pipeline de geração de conteúdo usando busca semântica em tempo real + modelo de linguagem generativo.

## Como funciona

1. Usuário fornece um tema
2. **Tavily API** realiza busca web em tempo real sobre o tema
3. **Gemini** processa os resultados e gera um post otimizado para redes sociais

## Stack

| Ferramenta | Função |
|------------|--------|
| Tavily API | Busca web semântica em tempo real |
| Google Gemini | Geração de texto com LLM |

## Objetivo

Projeto de aprendizado prático em automação com IA — primeiro de uma série focada em aplicações de dados e inteligência artificial para geração de insights e conteúdo.
